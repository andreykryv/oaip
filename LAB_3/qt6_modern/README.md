# Лабораторная работа №3 — Рекурсия (Qt 6 Modern)

## 📋 Описание

Современная реализация лабораторной работы №3 на **Qt 6** с обновлённым дизайном и улучшенной архитектурой.

### ✨ Особенности

- **Современный тёмный дизайн** с неоновыми акцентами
- **Qt 6.4+** с использованием CMake
- **C++17** стандарт
- **Плавные анимации** для Ханойской башни
- **Адаптивный интерфейс** с изменяемым размером окна

## 🚀 Сборка и запуск

### Требования

- Qt 6.4 или выше
- CMake 3.16+
- Компилятор с поддержкой C++17 (GCC 9+, Clang 10+, MSVC 2019+)

### Linux

```bash
cd LAB_3/qt6_modern
mkdir build && cd build
cmake ..
make -j$(nproc)
./bin/lab3_recursion
```

### Windows (MinGW)

```cmd
cd LAB_3\qt6_modern
mkdir build && cd build
cmake -G "MinGW Makefiles" ..
mingw32-make
bin\lab3_recursion.exe
```

### Windows (MSVC)

```cmd
cd LAB_3\qt6_modern
mkdir build && cd build
cmake -G "Visual Studio 17 2022" ..
cmake --build . --config Release
Release\lab3_recursion.exe
```

## 📚 Реализованные задания

| № | Задание | Описание |
|---|---------|----------|
| 1 | Dec → Bin | Перевод десятичного числа в двоичную систему |
| 2 | Аккерман | Функция Аккермана A(m,n) |
| 3 | Реверс числа | Рекурсивный переворот числа |
| 4 | Ханойская башня | Классическая задача с визуализацией |
| 5 | Файловая система | Рекурсивный обход директорий |

## 🎨 Дизайн

### Цветовая схема

- **Фон**: Тёмный сине-фиолетовый (#0f0f1a)
- **Акцент**: Фиолетовый неон (#7c3aed) + Циан (#06b6d4)
- **Текст**: Светло-серый (#f8fafc)

### Элементы UI

- Градиентные кнопки
- Карточки с закруглёнными углами
- Современный статус-бар
- Анимированные вкладки

## 🔧 Отличия от старой версии

| Характеристика | Старая (Qt 5) | Новая (Qt 6) |
|---------------|---------------|--------------|
| Система сборки | qmake (.pro) | CMake |
| Дизайн | Тёмная тема | Современный неон |
| Вкладки | Текст + иконки отдельно | Объединённые |
| Дерево файлов | setTextColor | setForeground |
