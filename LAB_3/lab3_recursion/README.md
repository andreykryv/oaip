### Qt 5/6 + C++17 | Графическое приложение с тёмной темой

---

## Структура файлов

```
lab3_recursion/
├── lab3.pro          ← Qt-проект
├── main.cpp          ← точка входа
├── mainwindow.h      ← заголовок главного окна
└── mainwindow.cpp    ← реализация всех 5 заданий
```

---

## Сборка и запуск

### Требования
- Qt 5.12+ или Qt 6.x
- Компилятор с поддержкой C++17 (GCC 7+, Clang 5+, MSVC 2019+)

### Windows (Qt Creator)
1. Открыть `lab3.pro` в Qt Creator
2. Нажать **Build** → **Run** (Ctrl+R)

### Linux / macOS (командная строка)
```bash
cd lab3_recursion
qmake lab3.pro
make -j$(nproc)
./lab3_recursion
```

### Windows (командная строка MinGW)
```cmd
cd lab3_recursion
qmake lab3.pro
mingw32-make
lab3_recursion.exe
```

## Реализованные задания

| # | Задание                         | Алгоритм                          | Тест         |
|---|---------------------------------|-----------------------------------|--------------|
| 1 | Dec → Bin                       | Рекурсивное деление на 2          | 66 → 1000010 |
| 2 | Функция Аккермана               | Взаимная рекурсия A(m,n)          | A(2,2) → 7   |
| 3 | Реверс числа                    | Арифметика + countDigits          | 972 → 279    |
| 4 | Ханойская башня                 | Классическая рекурсия hanoi(n)    | 3 кольца     |
| 5 | Обход файловой системы          | Рекурсивный QDir + QTreeWidget    | —            |

---

## Ключевые особенности

- **Тёмная тема** с акцентными цветами (фиолетовый + циан)
- **Шаговая визуализация** рекурсии для задач 1, 3
- **Справочная таблица** A(m,n) для задачи 2
- **Ограничение 10M вызовов** для Аккермана (защита от зависания)
- **QTreeWidget** с иконками для задачи 5
- **Открытие файлов** двойным кликом (QDesktopServices)
- **Статус-бар** с кратким результатом каждой задачи
